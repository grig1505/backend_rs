Реализация команды edit
Функция editNote должна:
Прочитать все заметки из db.json
Найти заметку с указанным ID
Если заметка найдена - обновить её title и сохранить
Если не найдена - вывести сообщение об ошибке (используя chalk.bgRed)
При успехе - вывести сообщение об успехе (используя chalk.bgGreen)

консольные команды:
Добавить: node index add --title="Hello"
Список: node index list
Удалить по ID: node index remove --id=1634833738358
Редактировать : node index edit --id "17652061282373" --title "Новое название заметки"
