Задача:
    Реализовать функционал по изменению названия заметки по ID
Тебе необходимо:
    Добавить команду edit, которая будет изменять заметку по ID
    Команды:
       index add  Add new note to list
       index list   Print all notes
       index remove Remove note by id
       index edit   Edit note by id

Команда принимает 2 параметра:
    --id - id заметки
    --title - новое название заметки
